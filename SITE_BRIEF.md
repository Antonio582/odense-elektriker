# Site Brief: odense-elektriker.dk

## Koncept
Lead generation site for en elektriker-virksomhed i Odense.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript
- **Animations:** Framer Motion
- **Deploy:** Vercel
- **Repo:** Antonio582/odense-elektriker

## Design: "Electric Precision"
Inspireret af Apple.com + Linear.app. Stram geometrisk grid.

### Farver
- Mørkeblå: #0a1628 (primary/baggrund)
- Elektrisk gul: #fbbf24 (accent/CTA)
- Hvid: #ffffff (tekst)
- Grå toner til sekundær tekst

### Font
- Inter (hele sitet)

### Effekter
- Subtle glow effects på CTAs
- Scroll-triggered reveals (Framer Motion)
- Parallax sections
- Counter animations
- Card hover effects
- Sticky scroll sections
- CTA pulse animation
- Gradient text
- Staggered grid animations
- Drag carousel
- Smooth accordion

## SEO
- **Primært keyword:** "elektriker odense" — 590 søgninger/md, CPC 5,43 kr
- **Sekundære:** akut elektriker odense, el-installatør odense, autoriseret elektriker odense, billig elektriker odense, elektriker fyn
- **Min 800 ord per side**
- **JSON-LD:** FAQPage + LocalBusiness + Breadcrumb på hver side
- **Min 3 interne links per side**
- **Meta titles & descriptions** optimeret for hvert keyword
